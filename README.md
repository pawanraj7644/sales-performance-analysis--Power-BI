# SuperStore Sales Dashboard (Power BI)

This project is a Power BI dashboard created using the SuperStore dataset. The main goal of this dashboard is to analyze sales performance, profit, and different business factors in a simple and interactive way.

## 📊 Dashboard Overview

The dashboard is divided into two pages:

### 🔹 Page 1: Sales Overview

* Summary of Total Sales, Quantity, and Profit (KPI cards)
* Sales distribution by Category and Sub-Category
* Region-wise filter (Central, East, South, West)
* Monthly Sales and Profit trends
* Segment and Payment Mode analysis
* Map visualization for location-based insights

### 🔹 Page 2: Detailed Analysis

* Sales trend based on Order Date
* State-wise sales performance
* Time-based analysis with zoom slider

## 🛠 Tools Used

* Power BI

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Use filters and visuals to explore the data
